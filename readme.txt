== Description ==

= WWD Lateral Info Plugin =

This plugin is intended to be used to access WWD (Westlands Water District)
lateral data by internal staff.  A public version of this plugin may be created
in the future to allow public to view some non critical data about systems.

== Installation ==

1. Download the plugin
2. Configure plugin by entering the API settings, role required for viewing content.
3. Create appropriate pages including short codes.  See the settings page for short code details.
